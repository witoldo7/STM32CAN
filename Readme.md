# STM32CAN - WQCAN
Idea of project is to provide CAN and BDM functionality for Saab Trionic 8 ECU. 

## Currently supported:
* CombiBeckend - For CAN operation work fine.
* SocketCAN - FDCAN and most of driver impl is done. Missing error reporting.
